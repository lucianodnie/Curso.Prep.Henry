bueno, para poder empezar a escribir códigos en javascript y poder plasmar tu creatividad para crear las aplicaciones que quieras.
es necesarios que entiendas estos simples conceptos:
las variables, las funciones, los strings, las declaraciones (if) y los valores booleanos(true, false).
las variables sirven para asginar un nombre "x" a un código o valor que quieras que sea representado por ese nombre, ya sea para resumir un codigo largo en una palabra o letra.
O también para mejor comprención del tanto como para vos como para el que vaya a leer la cadena de códigos que escribas.
por ej:
var a = 1;
var b = 2;
var c = a + b;
c = 3
o en un caso un poquito más complejo:
var d = document.getElementById;
Escribiendo la letra "d" nos ahorramos escribir todo el codigo "getElementBlablabla".
